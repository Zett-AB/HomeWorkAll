Поиск ошибок в коде с функциями JavaScript

В следующих задачах некоторый программист написал код и, возможно, допустил в нем ошибки.
Вы должны проверить, делает ли код то, что описано.
Если код работает некорректно, вы должны исправить ошибки. Всего 10 задач.

Задача № 1

Код должен найти сумму результатов работы двух функций:

function func1() {
console.log/alert(3);
}
function func2() {
console.log/alert(5);
}

console.log/alert( func1() + func2() );

Решение/правильный код смотрим в файле JS

В данном коде не будет работать последняя срока
консоль или вывод на экран будут выдывать NaN, поскольку внесенные числа не возвращаются в функции.
Для исправления добавляем в функцию return.

Задача № 2

Код должен найти сумму элементов массива:

function sum(arr) {
let res = 0;

for (let elem of arr) {
res += elem;
return res;
}
}

console.log/alert(sum([1, 2, 3, 4, 5]));

Решение/правильный код смотрим в файле JS

Выполнение данного кода всегда будет выводит 1, потому-что код будет выполнятся 1 раз.
Чтобы код выполнялся пока в массиве есть числа/значения, необходимо строку кода return res; вывести из цикла и вставить после него.
Тогда цикл for будет выполнятся столько раз сколько в массиве есть значений.

Задача № 5

Код должен найти сумму элементов массива:

let sum = sum([1, 2, 3, 4, 5]);
console.log/alert(sum);

function sum(arr) {
let sum = 0;

for (let elem of arr) {
sum += elem;
}

return elem;
}

Решение/правильный код смотрим в файле JS

В данном коде сразу несколько ошибок:
первая - название функции и переменной одинаковые, поэтому правим название переменной(но вы можете исправить название функции, но помните, что название функции должно отражать её суть);
вторая - в предпоследней строке кода нужно изменить elem на sum, поскольку если функция будет возвращать elem, то она не будет считать сумму массива.

Задача № 9

Код должен найти сумму цифр числа:

let num = 12345;
let res = getDigitsSum(num);
console.log/alert(res);

function getDigitsSum(num) {
let arr = num.split('');
let sum = 0;

for (let elem of arr) {
sum += elem;
}

return sum;
}

Решение/правильный код смотрим в файле JS

В данном задании, несмотря на простоту не все так просто.
У нас есть переменная num имеющая значением число 12345, для того чтобы слодить сумму цифр числа, на первоначально необходимо преобразовать число в строку.
Для этого мы в функции первой строкой прописываем num=String(num);.
Таким образом, мы преобразовали наше число в строку.
Далее мы преобразуем нашу строку в массив let arr = num.split('');.
Теперь нам нужно написать цикл, в течении которого будет происходить сложение значений массива и в далее в самом цикле мы прописываем уже непосредственно суммирование, которое будет выполнятся до тех пор пока будет выполнятся цикл.
